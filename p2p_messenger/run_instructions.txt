How to Run the UDP P2P Messenger

This guide explains how to install and run the application on three different operating systems: Windows, macOS, and Linux.

Prerequisites for all systems:
- Python 3.7 or higher installed.

---

1. WINDOWS

Step 1: Open Command Prompt or PowerShell and navigate to the project directory (where app.py is located):
> cd path\to\p2p_messenger

Step 2: (Optional but recommended) Create and activate a virtual environment:
> python -m venv venv
> venv\Scripts\activate

Step 3: Install the required dependencies:
> pip install -r requirements.txt

Step 4: Run the application:
> python app.py

---

2. macOS

Step 1: Open Terminal and navigate to the project directory:
$ cd path/to/p2p_messenger

Step 2: (Optional but recommended) Create and activate a virtual environment:
$ python3 -m venv venv
$ source venv/bin/activate

Step 3: Install the required dependencies:
$ pip3 install -r requirements.txt

Step 4: Run the application:
$ python3 app.py

---

3. LINUX (Ubuntu/Debian-based)

Step 1: Ensure Python 3, pip, and the Tkinter system library are installed. Open Terminal and run:
$ sudo apt update
$ sudo apt install python3 python3-pip python3-venv python3-tk

Step 2: Navigate to the project directory:
$ cd path/to/p2p_messenger

Step 3: (Optional but recommended) Create and activate a virtual environment:
$ python3 -m venv venv
$ source venv/bin/activate

Step 4: Install the required dependencies:
$ pip install -r requirements.txt

Step 5: Run the application:
$ python3 app.py
